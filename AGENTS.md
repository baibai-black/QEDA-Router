# AGENTS

- Do not commit any real chip layouts, PDK files, process decks, or private data.
- Examples must stay toy-scale and clearly non-production.
- Do not claim this project can be used directly for tape-out or fabrication.
- Run `pytest` after every code change.
- Keep modules independent instead of collapsing logic into one large script.
- Preserve DRC warnings in outputs; do not hide or auto-dismiss them.
- CoC has higher priority than TWL in optimization decisions.
- This tool is for learning QEDA auto-routing ideas and does not replace professional layout tools or human review.
